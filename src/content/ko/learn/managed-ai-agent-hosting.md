# 안전한 에이전트 플릿을 위한 매니지드 AI 에이전트 호스팅

여러분이 원한 것은 AI 인력이었습니다. 받은 것은 DevOps라는 두 번째 직업이었습니다. 매니지드 AI 에이전트 호스팅은 그 직업을 돌려줍니다. 여러분의 에이전트가 필요로 하는 서버, 컨테이너, 자격 증명 볼트, 예산을 운영하므로, 여러분은 그 무엇도 직접 프로비저닝하지 않습니다. OpenLegion의 매니지드 플레인은 각 에이전트를 전용 VPS의 자체 격리 컨테이너에 떨어뜨리고, 에이전트가 결코 건드리지 않는 볼트에 여러분의 키를 보관하며, 기본적으로 에이전트별 지출을 제한합니다. 여러분은 목표와 키를 가져오고, 그 아래 바닥은 우리가 운영합니다.

> **매니지드 AI 에이전트 호스팅이란 무엇입니까?**
> 매니지드 AI 에이전트 호스팅은 자율 AI 에이전트가 실행되는 인프라, 즉 컨테이너, 자격 증명 저장소, 예산, 네트워킹, 제어 대시보드를 프로비저닝하고, 격리하고, 보안하고, 운영하는 서비스입니다. 그래서 팀은 그 인프라를 직접 구축하거나 유지하지 않고도 에이전트 플릿을 배포할 수 있습니다.

## 핵심 요약

- 매니지드 AI 에이전트 호스팅은 여러분과 실행 중인 플릿 사이의 작업을 제거합니다: 서버 설정도, Docker 튜닝도, 운영할 볼트도 없습니다.
- 모든 에이전트는 공유 멀티 테넌트 프로세스가 아니라 전용 VPS의 자체 컨테이너에서 실행되므로, 한 계정의 폭발 반경이 격납된 채로 유지됩니다.
- 자격 증명은 신뢰된 호스트의 볼트 프록시에 머뭅니다. 에이전트가 요청을 보내면 프록시가 네트워크 레이어에서 키를 주입하므로, 탈취된 에이전트는 키를 읽을 수 없습니다.
- 에이전트별 일별 및 월별 예산 상한이 폭주하는 LLM 지출을 달러 단위로 멈추며, 모델 사용에 마크업이 없습니다.
- 동일한 엔진이 BSL 1.1 하에 소스가 공개되어 있으므로, 매니지드로 시작해 나중에 재작성 없이 셀프 호스팅으로 옮길 수 있습니다.
- 플랜은 월 $19부터 시작하며, 첫날부터 결제되고, 7일 환불 보장이 있습니다.

## "그냥 에이전트 하나 실행하기"의 숨은 비용

에이전트 하나를 세우는 일은 주말입니다. 신뢰할 수 있는 플릿을 운영하는 일은 한 분기입니다.

프로토타입과 프로덕션 사이 어딘가에서, 작업은 조용히 모양을 바꿉니다. 프롬프트를 튜닝하기를 멈추고 컨테이너를 튜닝하기 시작합니다. 에이전트가 무슨 말을 해야 하는지를 생각하기를 멈추고, 그것의 API 키가 어디 있는지, 새벽 3시에 하나가 루프에 빠지면 무슨 일이 일어나는지, 탈취된 에이전트가 나머지 아홉에 도달하지 못하게 어떻게 막을지를 생각하기 시작합니다. 그 어느 것도 여러분이 하려던 일이 아닙니다. 그 전부가 에이전트를 켜둔 채로 두어도 안전한지를 결정하는 일입니다.

매니지드 호스팅은 바로 그 레이어를 흡수하기 위해 존재합니다. OpenLegion의 호스팅 제공물로 여러분은 다음을 얻습니다:

- **계정별로 프로비저닝되는 전용 VPS**, 그래서 여러분의 플릿은 누구의 것과도 프로세스나 메모리 공간을 결코 공유하지 않습니다.
- **에이전트별 컨테이너 격리**, 리소스 상한, 비루트 실행, 권한 제거, 기본 읽기 전용 파일시스템을 갖춤.
- **자격 증명 볼트 프록시**, LLM 키, OAuth 토큰, 지갑 개인 키를 에이전트 컨테이너 밖에 완전히 보관.
- **에이전트별 예산 시행**, 하드 컷오프와 함께, 그래서 루프에 빠진 에이전트가 깜짝 청구서를 만들 수 없음.
- **제어 대시보드**, 템플릿을 배포하고, 에이전트와 채팅하고, 라이브 비용과 상태를 지켜보고, 오작동하는 것을 일시 중지.
- **환영 LLM 크레딧**, 모든 유료 플랜에 제공되며, 100+ 제공자 전반에 마크업 제로로 본인의 키를 가져오는 옵션도 함께.

## 매니지드 호스팅 vs 셀프 호스팅: 선택하는 법

OpenLegion은 동일한 엔진을 두 가지 방식으로 제공하므로, 결정은 어느 버전이 더 유능한지가 아니라 누가 박스를 운영하는지에 관한 것입니다.

| 고려 사항 | 셀프 호스팅 (BSL 1.1) | 매니지드 호스팅 |
|---|---|---|
| 누가 서버를 운영하는가 | 여러분 | OpenLegion |
| 설치 시간 | 여러분 머신에서 세 명령 | 가입하고 템플릿 선택 |
| 자격 증명 볼트 | 여러분이 운영 | 신뢰된 호스트에서 대신 운영 |
| 업데이트와 패치 | 여러분이 풀하고 재배포 | 대신 적용 |
| 데이터 거주 | 전적으로 여러분 인프라에 | 우리가 프로비저닝하는 전용 VPS에 |
| 적합한 용도 | 규제, 에어 갭, 완전한 통제 | 오늘 당장 플릿을 돌리려는 팀 |
| 비용 모델 | 여러분 자신의 서버 비용 | 월 $19부터 시작하는 정액 플랜 |

컴플라이언스, 데이터 거주, 또는 호스트에 대한 전적인 통제가 그것을 운영하는 비용을 능가할 때 셀프 호스팅을 선택하십시오. 오늘 오후 에이전트가 일하기를 원하고 볼트, 컨테이너 런타임, 업데이트 주기를 소유하지 않으려 할 때 매니지드 호스팅을 선택하십시오. 둘 다 동일한 코드를 실행하므로, 매니지드로 시작해도 나중에 셀프 호스팅의 길이 막히지 않습니다.

## 배포를 누르면 일어나는 일

유료 플랜을 활성화하면 OpenLegion이 전용 VPS를 프로비저닝하고 메시 호스트를 온라인으로 가져옵니다. 오퍼레이터 에이전트가 여러분 플릿의 작업반장으로 자동 생성됩니다. 거기서부터 여러분이 원하는 팀, 즉 리서치 데스크, 영업 파이프라인, 콘텐츠 스튜디오를 설명하면, 플랫폼이 각 역할을 자체 메모리, 예산, 도구 권한을 갖춘 자체 컨테이너에 배포합니다.

에이전트는 [오케스트레이션 레이어](/learn/ai-agent-orchestration), 즉 SQLite 기반 블랙보드, pub/sub 이벤트 버스, 구조화된 핸드오프를 통해 조율합니다. 라우팅을 결정하는 언어 모델이 컨트롤 플레인에 앉아 있지 않으므로, 동작이 감사 가능하고 비용이 예측 가능하게 유지됩니다. 여러분은 대시보드를 통해, 또는 Telegram, Discord, Slack, WhatsApp, 웹훅을 통해 에이전트와 대화합니다.

## 직접 구축하지 않아도 되는 보안

에이전트를 안전하게 호스팅하는 가장 어려운 부분은 팀이 가장 자주 건너뛰는 부분인데, 실패하기 전까지 보이지 않기 때문입니다. OpenLegion의 매니지드 플레인은 [AI 에이전트 보안](/learn/ai-agent-security) 모델에 설명된 것과 동일한 심층 방어를 기본으로 활성화한 채 시행합니다:

- 공유 프로세스 공간 없이, 에이전트별 컨테이너 격리.
- 네트워크 레이어에서 자격 증명을 주입하는 볼트 프록시, 그래서 에이전트는 원시 키를 결코 쥐지 않음.
- 각 에이전트가 어떤 도구, 파일, 작업을 사용할 수 있는지를 다스리는 에이전트별 권한 매트릭스.
- 모든 경계에서의 입력 새니타이즈, SSRF 보호, 프롬프트 인젝션 하드닝.
- 닫힘으로 실패하는 에이전트별 예산 상한.

여러분은 하드닝 가이드를 읽고 직접 구현하는 것이 아니라 가입함으로써 이것들을 물려받습니다. 그 아래 런타임은 [AI 에이전트 플랫폼](/learn/ai-agent-platform) 개요를 참조하십시오.

## 자주 묻는 질문

### 매니지드 AI 에이전트 호스팅이란 무엇입니까?

매니지드 AI 에이전트 호스팅은 자율 AI 에이전트가 필요로 하는 인프라, 즉 격리된 컨테이너, 자격 증명 볼트, 예산 시행, 네트워킹, 제어 대시보드를 프로비저닝하고 운영하는 서비스입니다. 그래서 팀은 그 인프라를 구축하거나 유지하지 않고도 에이전트 플릿을 배포하고 실행할 수 있습니다. OpenLegion에서는 가입하고, 템플릿을 고르고, LLM 키를 추가하면, 여러분의 에이전트가 전용 VPS에서 실행됩니다.

### 매니지드 호스팅은 OpenLegion 셀프 호스팅과 어떻게 다릅니까?

둘 다 동일한 보안 통제를 갖춘 동일한 OpenLegion 엔진을 실행합니다. 셀프 호스팅은 여러분이 서버를 운영하고, 볼트를 운영하고, 업데이트를 직접 적용하는 것을 의미하며, 규제되거나 에어 갭 환경에 적합합니다. 매니지드 호스팅은 OpenLegion이 VPS를 프로비저닝하고, 볼트를 운영하며, 플랫폼을 대신 패치하는 것을 의미합니다. 코드가 동일하므로, 매니지드로 시작해 나중에 플릿을 재구축하지 않고 셀프 호스팅으로 마이그레이션할 수 있습니다.

### 내 API 키와 지갑 키를 쥐는 AI 에이전트를 호스팅해도 안전합니까?

예, 키가 에이전트 안에 결코 존재하지 않을 때 그렇습니다. OpenLegion의 매니지드 플레인에서 API 키, OAuth 토큰, 지갑 개인 키는 신뢰된 메시 호스트의 볼트 프록시에 보관됩니다. 에이전트가 요청을 보내면 프록시가 네트워크 레이어에서 자격 증명을 주입하므로, 완전히 탈취된 에이전트조차 비밀을 읽거나 외부로 빼낼 수 없습니다. 지갑 트랜잭션은 서버 측에서 서명됩니다.

### 매니지드 AI 에이전트 호스팅의 비용은 얼마입니까?

OpenLegion 매니지드 플랜은 월 $19부터 시작하며, 첫날부터 결제되고, 7일 환불 보장이 있습니다. 정액 요금은 전용 VPS, 볼트 프록시, 컨테이너 프로비저닝, 대시보드, 그리고 결코 만료되지 않는 환영 LLM 크레딧 묶음을 포함합니다. 모델 토큰 사용은 그 크레딧에서 차감되거나 여러분 자신의 제공자가 OpenLegion의 마크업 없이 직접 청구합니다.

### 매니지드 호스팅을 사용하려면 Docker나 DevOps를 알아야 합니까?

아니오. 매니지드 플레인이 컨테이너 프로비저닝, 자격 증명 저장, 네트워킹, 업데이트를 처리합니다. 여러분은 대시보드를 통해 작업합니다: 팀 템플릿을 고르고, LLM 키를 추가하고, 배포합니다. Docker, Python, 서버 관리는 대신 셀프 호스팅 버전을 선택할 때만 필요합니다.

### 나중에 매니지드 호스팅에서 셀프 호스팅으로 옮길 수 있습니까?

가능합니다. 매니지드 플레인과 셀프 호스팅 배포판은 BSL 1.1 하에 소스가 공개된 동일한 엔진을 실행합니다. 마이그레이션을 막을 독점 락인 레이어가 없으므로, 팀은 흔히 빠르게 움직이기 위해 매니지드 호스팅으로 시작했다가 컴플라이언스나 비용이 그럴 가치가 있게 만들면 셀프 호스팅 인프라로 전환합니다.

### 매니지드 플랜에서 에이전트를 몇 개나 호스팅할 수 있습니까?

에이전트 한도는 플랜 티어에 따라 확장되며, 엔트리 플랜의 단일 에이전트부터 상위 티어의 대규모 플릿까지, 엔터프라이즈를 위한 커스텀 한도와 함께입니다. 배포된 각 에이전트 컨테이너가 한도에서 한 에이전트로 계산되므로, Dev Team 같은 세 역할 템플릿은 세 에이전트로 계산됩니다.
